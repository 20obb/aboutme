HiLoader

بوت Telegram لتحميل محتوى من السوشيال ميديا. مشروع بسيط وعملي كبداية قوية.

اقرأ المزيد (README)

GNU Loader Tweak (iPhone)

نسخة مفتوحة المصدر معدّلة من TikTok لأجهزة iOS.

مزايا رئيسية:
  • تحميل الفيديوهات والصوتيات (MP4, MOV) بدقة تصل إلى 4K.
  • تشغيل في الخلفية (Background Play).
  • استخراج الرسائل والردود المخفية بدون علم الطرف الآخر وتصديرها PDF.
  • تغيير الموقع الجغرافي/البلد الافتراضي عبر واجهة سهلة أو بإدخال إحداثيات يدوياً.
  • وضع “التصفح المتخفّي” Stealth Mode لمنع تسجيل التفاعلات.
  • تكامل مع Siri Shortcuts.
  • سمة Red Hat (Dark Theme) متوافقة مع iOS Dark Mode.

التقنيات: Swift, UIKit, Alamofire, CoreLocation & VPN Framework, SQLite/Realm.

اقرأ المزيد (README)

Evilgram (iPhone)

نسخة Telegram بشبه TikTok للأيفون.

مزايا: عرض الرسائل المحذوفة، عرض الصور ذاتية التدمير، واجهة “For You”، تشغيل الفيديو/الصوت في الخلفية.

التقنيات: Swift, Telegram API, CocoaPods.

اقرأ المزيد (README)

Web Privacy Insights

أداة ويب لتحليل خصوصية المواقع.

وظائف: جمع وتحليل الكوكيز، بصمة JavaScript، رؤوس HTTP. استخدام FastAPI للـ Back-end وTotalVirus API لفحص الروابط. إصدار تقرير أمان (Security Score).

التقنيات: Python, FastAPI, PostgreSQL, Docker.

اقرأ المزيد (README)

Fetcher OSINT

أداة OSINT مفتوحة المصدر للبحث عن المستخدمين.

  • بحث عادي: معلومات عامة عن الحساب (بروفايل، تاريخ التفعيل).
  • بحث عميق: استخراج سجل الرسائل والتفاعلات الكاملة.

التقنيات: Python, Requests, BeautifulSoup, SQLite.
